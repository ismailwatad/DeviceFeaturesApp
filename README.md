## Kurulum
1. Depoyu klonlayın veya indirin.
2. Proje dizinine gidin: `cd DeviceFeaturesApp`.
3. Bağımlılıkları kurun: `npm install`.
4. Uygulamayı başlatın: `npx expo start` ve Expo Go ile QR kodunu okutun.

## Özellikler
- Kamera ile fotoğraf çekme
- Galeriden fotoğraf seçme
- Seçilen resmi görüntüleme
- GPS konumunu alma
- Yerel bildirim tetikleme
- Başarılı işlemlerde dokunsal geri bildirim

## Test Kontrol Listesi
- Kamera/Galeri izinleri isteniyor
- Seçimden sonra resim görüntüleniyor
- GPS izni isteniyor ve koordinatlar gösteriliyor
- Bildirim gösteriliyor
- Başarılı işlemde haptics tetikleniyor
- Reddedilen izinlerde çökme olmuyor
